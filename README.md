# Simple Sign-In Page

## Overview
This is a simple sign-in page built using HTML and CSS. It provides a basic form for users to enter their login details, including username, mobile number, happiness status, and password.

## Features
- **Responsive Navigation Bar**: Includes links for About, Advise, and Services.
- **User-Friendly Form**: Fields for username, mobile number, happiness status, and password.
- **Minimalist Design**: Simple and clean UI.

## Technologies Used
- **HTML**: Structure of the page
- **CSS**: Styling and layout (linked as `style.css`)

## How to Use
1. Open the `index.html` file in a web browser.
2. Enter the required details in the form fields.
3. Click on the **Sign In** button.
